# Разработка системы распознавания эмоций в реальном времени
Проект создан в рамках курса "Машинное обучение и интеллектуальные системы". Создана модель, обученная определять эмоции по видео, и прототип системы, который определяет эмоции в реальном времени.

## Описание проекта
В условиях роста популярности видеосвязи (Zoom, Microsoft Teams, Skype) теряется часть невербальной коммуникации. Автоматическое распознавание эмоций позволяет:

* улучшить качество удалённого взаимодействия;

* помочь в телемедицине, образовании, клиентском сервисе;

* создать основу для адаптивных интерфейсов.

**Цель работы** – разработать сервис, который в реальном времени анализирует эмоции человека по видеопотоку.
В проекте решаются следующие **задачи**:

* выбор и подготовка датасета (AffectNet в формате YOLO);

* обучение модели YOLOv8 для классификации 8 эмоций;

* интеграция модели с видеопотоком (веб-камера) и построение эмоционального профиля.

## Результат

В результате разработки были созданы:

* Ноутбук в Google Colab, содержищй полное описание проекта и обучение модели.
* Прототип, созданный в PyCharm, который реализует распознавание эмоций в реальнмо времени по веб-камере.
